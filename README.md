# ft_contrast

Applying contrast to a picture in the .pgm format. Using parallele threads to do the contrast.

Compiling as:
```ft_contrast -f image.pgm -c 12 -o pouet.pgm```

# Allowed functions

* free
* open
* close
* write
* read
* lseek
* read
* fork
* printf function family (see man 3 printf)
* pthread_create
* pthread_cleanup_push
* pthread_detach
* pthread_exit
* pthread_join
* pthread_mutex_init
* pthread_cond_init
* pthread_cond_wait
* pthread_cond_signal
